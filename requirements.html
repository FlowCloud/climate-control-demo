Software Requirements

Download MPLAB IPE v3.00 from http://microchip.wikidot.com/ipe:installation This IPE will be required to program the embedded hex file on the Wi-Fire boards. There are separate sections for installing on Windows and Linux. Please follow as per your host pc.
